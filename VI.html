Virtual Assistant

Overview

The Virtual Assistant is a Python-based voice assistant that performs tasks like opening websites, fetching time, and searching info. It uses speech recognition and text-to-speech for natural interaction. This project showcases real-time voice control using Python libraries.

Key Features

  • Voice command recognition using SpeechRecognition library
  • Text-to-speech responses via pyttsx3
  • Performs actions like opening websites, telling time, and searching Wikipedia
  • Real-time interaction with natural language input

Technology Used

  • Python
  • pyttsx3 — For text-to-speech conversion
  • SpeechRecognition— Converts voice to text
  • smtplib
  • webbrowser
  • wikipedia
