Web URL Analyzer

Overview

The URL Analyzer is a Python-based web application that analyzes and classifies URLs to determine whether they are legitimate, suspicious, or phishing. This tool is designed to enhance cybersecurity awareness by helping users identify potentially malicious URLs before clicking on them.

Key Features

URL Feature Extraction includes checking for:

  • Presence of IP address
  • URL length
  • Use of "@" symbol
  • Redirection "//" usage
  • HTTPS presence
  • Domain age
  • Prefix/suffix in domain
  • And many more

Machine Learning Model

Model Used: Random Forest Classifier

The Random Forest Classifier is an ensemble algorithm that builds multiple decision trees and combines their results for better accuracy. It uses random subsets of data and features to avoid overfitting. Final output is based on majority vote. It's reliable and effective for classification tasks.

I implemented this model to classify URLs as phishing or legitimate. The model was trained using a dataset of labeled URLs by extracting features such as URL length, presence of IP address, use of "@" symbol, etc. I split the dataset into training and test sets, trained the model using RandomForestClassifier from Scikit-learn, and evaluated its performance.
