Assistive Technology for Dyslexia

Overview

The project aims to support individuals with dyslexia by offering a digital platform that improves reading, spelling, and comprehension. It integrates speech-to-text and text-to-speech technologies with real-time PDF customization, allowing users to personalize fonts, spacing, and letter differentiation. Designed using Python and Flask, the tool enhances accessibility and usability, helping dyslexic learners interact with educational materials effectively and independently.

Key Features

  • Speech-to-Text (STT): Allows users to dictate content, reducing typing challenges
  • Text-to-Speech (TTS): Reads out text using customizable voice settings (via pyttsx3)
  • PDF Customization: Modify font size, line height, spacing, and letter styles (e.g., distinguishing ‘b’ from ‘d’) using PDFTron
  • Focus Tracker: A visual ‘L’-shaped guide helps users track lines while reading
  • Multi-Sensory Learning: Combines visual and auditory modes for better comprehension

Technology Used

  • HTML
  • CSS
  • JavaScript
  • Python
  • pyttsx3 — For text-to-speech conversion
  • SpeechRecognition— Converts voice to text
  • PDFTron — For dynamic PDF rendering and modification
  • Framework: Flask
