Stock Data Extractor

Overview

This project performs a comparative analysis of Tesla (TSLA) and GameStop (GME) stock prices over a selected period using the yfinance and plotly libraries. The goal is to determine which stock performed better in terms of price return from 2020 to 2023 by plotting both on a graph and computing their respective percentage returns.

Technology Used

  • Python
  • Jupyter Notebook (development environment)
  • yfinance - for retrieving historical stock data
  • plotly - for creating interactive visualizations
  • pandas - for data handling and manipulation

Output

After fetching the data, the script calculates the percentage growth of each stock from the beginning to the end of the period and visualizes the stock price trends. The analysis concludes that:

Tesla (TSLA) performed better than GameStop (GME) in terms of overall stock price return during the analyzed period.
